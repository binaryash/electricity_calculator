# Documentation Index

This folder documents how the Electricity Use Planner works from both a product and implementation perspective.

## Documents

- [Calculation Model](./calculation-model.md)
  Explains the budget math, reserve logic, daily-cap derivation, pacing metrics, and selected-set balancing.

- [Appliance Planner](./appliance-planner.md)
  Explains appliance assumptions, default selections, runtime sliders, and how selected appliances are turned into a safe daily plan.

- [Development Guide](./development-guide.md)
  Explains project structure, important files, persistence behavior, and how to extend the app safely.

## Recommended Reading Order

If you are using the app:

If you are modifying the app:
